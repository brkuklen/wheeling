# Wheeling

A short, free reference for the **Wheel** options-income strategy (cash-secured puts +
covered calls). Two self-contained HTML pages, no dependencies, no tracking.

**Live site:** _will be filled in after first GitHub Pages build_

## Contents

| File | What |
|------|------|
| [`index.html`](index.html) | Landing page linking to the two below |
| [`wheel_guide.html`](wheel_guide.html) | Long-form mechanical guide: entries, rolls, assignment, math |
| [`wheel_training_quest.html`](wheel_training_quest.html) | Interactive four-phase quest map with progress tracking |

## Running locally

Open any of the HTML files directly in a browser. There is no build step, no server, no
JavaScript dependencies.

```bash
# macOS
open index.html

# Windows (PowerShell)
Start-Process .\index.html
```
